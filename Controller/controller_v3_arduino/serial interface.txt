MACHINE  DATA  FORMAT

MATLAB   "1"     binary
ARDUINO  "1"     binary

ARDUINO  PT1\n   ASCII raw int newline terminated
ARDUINO  PT2\n   ASCII raw int newline terminated
ARDUINO  PT3\n   ASCII raw int newline terminated
ARDUINO  SG1\n   ASCII raw int newline terminated
ARDUINO  SG2\n   ASCII raw int newline terminated
ARDUINO  SG3\n   ASCII raw int newline terminated
